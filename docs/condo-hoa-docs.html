Condo/HOA Document Review — What to Look For

Use the document window wisely; many forms allow cancellation if documents are unacceptable within time.

Checklist

  • Budgets & reserves (adequate?)
  • Pending special assessments
  • Major litigation
  • Insurance coverages
  • Rules & leasing policy

Tip

Ask for board meeting minutes for upcoming projects. Calendar the last day to cancel if docs are unacceptable.
